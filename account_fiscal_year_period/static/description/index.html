Año Fiscal y Periodo Contable

  • Funcionalidad :
    • Permitir al usuario definir el Año Fiscal.
    • Permitir al usuario generar Periodos para el Año fiscal (por mes).
    • Permitir al usuario abrir o cerrar cualquier Periodo del mes.
    • Evitar un registro posterior si la fecha de entrada tiene:
    • Año Fiscal no definido
    • No definido el Periodo para el Año Fiscal
    • Período con Año fiscal no abierto
    • Periodo de mes cCerrado
    • Definir Acceso de Usuario:
    • Definir Años Fiscal : Crear registro de Período de Año fiscal.
    • Crear Periodos : Crear Periodos del Año Fiscal.
    • Cerrar Periodo : Cerrar Periodos Contables.
    • Apertura/Cierre de mes: Abrir o cerrar cualquier mes del Periodo.
